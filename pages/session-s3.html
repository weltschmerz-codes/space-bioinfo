2.1 Categories of Bioinformatics Tools

Bioinformatics tools can be organized according to the stage of analysis they address. Most RNA-seq-based analyses follow a standard workflow from raw data to biological interpretation.

2.2 Sequence Alignment Tools

Sequence alignment is one of the most fundamental operations in bioinformatics. It is the process of comparing two or more sequences to find regions of similarity. Almost every downstream analysis — from finding which genome a read came from to identifying homologous proteins — relies on alignment.

2.2.1 BLAST (Basic Local Alignment Search Tool)

BLAST is the most widely used bioinformatics tool in history. It searches a query sequence (DNA or protein) against a database to find similar sequences. Key BLAST variants: blastn (nucleotide vs nucleotide), blastp (protein vs protein), blastx (translated nucleotide vs protein database).

💡 BLAST E-value Explained

The E-value (expect value) is the number of hits you would expect to see by chance with a given score. An E-value of 0.001 means you would expect 0.001 random hits of that score in a database of this size — meaning the hit is almost certainly real. A rule of thumb: E < 1e-5 is significant; E < 1e-10 is strong; E < 1e-50 means very high sequence similarity.

2.2.2 STAR and HISAT2: RNA-seq Aligners

For RNA-seq, short reads must be aligned to a reference genome in a splice-aware manner — meaning the aligner must recognize that a read might span an exon-exon junction. Standard aligners like BWA cannot do this. STAR (Spliced Transcripts Alignment to a Reference) and HISAT2 are the two most commonly used splice-aware aligners. STAR is typically faster and preferred for large genomes. HISAT2 uses less memory, making it suitable for servers with limited RAM.

2.3 Differential Gene Expression Tools

ToolLanguageStatistical ModelBest Used For

2.4 Gene Ontology and Pathway Analysis Tools

Once a list of DEGs is obtained, the next step is biological interpretation. A list of gene names is not directly interpretable — we need to know what biological processes, molecular functions, or cellular compartments those genes are enriched in. This is the role of Gene Ontology (GO) analysis and pathway enrichment analysis.

ToolAccessBest Feature
DAVIDWeb (david.ncifcrf.gov)Simple web interface; good for quick annotation and pathway clustering
g:ProfilerWeb + R packageExcellent for multi-organism support; regularly updated databases
GSEADesktop app + RGene set enrichment without arbitrary fold-change cutoffs; considers rank of all genes
clusterProfilerR (Bioconductor)Comprehensive R-based enrichment; integrates with ggplot2 for publication figures
EnrichrWeb (maayanlab.cloud/Enrichr)Fast; over 200 gene set libraries; accepts gene lists directly

Session Summary

  • Bioinformatics tools are organized by pipeline stage: QC → alignment → quantification → differential expression → pathway analysis
  • BLAST performs sequence similarity searches; STAR and HISAT2 perform splice-aware RNA-seq alignment
  • DESeq2, edgeR, and limma-voom are the three major differential expression tools; DESeq2 is preferred for small sample sizes
  • GO and pathway tools (DAVID, g:Profiler, GSEA, clusterProfiler) translate gene lists into biological meaning
  • FASTQ → SAM/BAM → count matrix → CSV/TSV is the standard file progression in an RNA-seq pipeline
