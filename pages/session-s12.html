📋 About This Worksheet

This guided practice session takes you through the NASA Open Science Data Repository (OSDR) using real mouse transcriptomic datasets. You will navigate the live database, inspect metadata, evaluate datasets for pipeline compatibility, and explore processed results through the Visualization Portal — all without writing a single line of pipeline code.

Complete all 16 tasks across five parts. Your answers are saved locally in your browser — click Save at any time. Tasks are marked complete when you tick the checkbox on each card. Some tasks have hints if you get stuck.

Estimated time: 90 – 120 minutes  ·  You will need: A web browser and access to osdr.nasa.gov

3f. If you were running the Session 9 pipeline on a new dataset with a read length of 100 bp, what value would you set for STAR's --sjdbOverhang parameter, and why?
The formula is: sjdbOverhang = read_length − 1. For 100 bp reads, sjdbOverhang = 99. This tells STAR the maximum possible overhang on each side of a splice junction.

12c. What does this result imply for multi-tissue studies? Should you always analyze different tissues together in one DESeq2 run?
When tissue dominates the PCA, it means the gene expression differences between liver and spleen are much larger than the differences between FLT and GC within any single tissue. Mixing tissues in one DESeq2 run would confound the analysis — it's almost always better to analyze each tissue separately.

Bioinformatics Pitfalls — Spot the Problem
Critical evaluation · 8 min

Each scenario below describes a researcher making a mistake when working with OSDR datasets. Identify the problem and explain how you would fix it.

Scenario A: A researcher downloads a dataset with n = 2 samples per group and proceeds to run DESeq2. The results show 8,000 differentially expressed genes with padj < 0.05.
Scenario B: A researcher wants to compare spaceflight effects across liver and spleen simultaneously. They merge all samples from OSD-245 and OSD-246 into one DESeq2 run with a simple FLT vs. GC design.
Scenario C: A researcher re-processes raw FASTQs from OSD-173 using STAR version 2.7.11a, but the original GeneLab processing used STAR 2.7.6a. They report that their counts differ significantly from the official GeneLab counts.

Multi-Tissue Hypothesis — Systemic vs. Tissue-Specific Response
Scientific synthesis · 10 min

One of the most important questions in spaceflight biology is whether the transcriptional response to microgravity is systemic (the same genes change in every tissue) or tissue-specific (different genes change in different tissues). OSDR datasets like the RR-6 panel make this comparison possible.

15a. What genes or biological processes would you expect to respond systemically to microgravity, regardless of tissue? Give at least two examples with your reasoning.
15b. What genes or processes would you expect to be tissue-specific? Give at least one example each for liver and skeletal muscle.
15c. If you were to test this hypothesis using OSDR data, describe your analysis strategy in 3–4 sentences. Which datasets would you use and what would you look for?
