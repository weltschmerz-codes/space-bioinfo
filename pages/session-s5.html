5.1 NASA OSDR: Background and Mission

NASA OSDR, incorporating GeneLab (osdr.nasa.gov) was established in 2015 as the first comprehensive open-science platform for spaceflight omics data. It is now part of the NASA Open Science Data Repository (OSDR), accessible at osdr.nasa.gov.

Before GeneLab, spaceflight omics data existed in fragmented forms across individual laboratory repositories and published supplementary materials, often without the detailed metadata needed to properly interpret the results. GeneLab changed this by creating a single portal with rich metadata capture using standard ontologies and providing both raw and processed data for every dataset.

As of 2024, OSDR hosts over 500 studies, including close to 1,000 spaceflight or space-analog datasets, covering organisms including Homo sapiens, Mus musculus, Rattus norvegicus, Arabidopsis thaliana, Drosophila melanogaster, Caenorhabditis elegans, and microbial species.

4.2 GeneLab Data Structure

Every GeneLab dataset is assigned a unique GeneLab Accession Number (GLDS-XXXX or OSD-XXXX). GeneLab uses a modified version of the ISA (Investigation/Study/Assay) framework, an internationally recognized standard for organizing life science experimental data.

4.2.1 Study Metadata

The study metadata describes: Study title, abstract, and objectives; organism(s) studied and strain/genotype information; study factors (the experimental variables, e.g., spaceflight vs. ground control); sample metadata (tissue, sex, age, treatment); assay types (what omics data was generated); and protocol references.

4.2.2 Data Files

Data LevelContents
Raw dataFASTQ files (for RNA-seq), raw mass spectrometry files, raw microarray files — straight from the instrument
Processed dataTrimmed FASTQ files, aligned BAM files, gene count matrices (counts per gene per sample)
Differential expression resultsDESeq2 or limma outputs: log2 fold changes, p-values, adjusted p-values for all genes
Visualization filesPCA plots, heatmaps, volcano plots generated by the GeneLab pipeline
Metadata filesISA-formatted sample and assay tables; essential for correctly assigning samples to experimental groups

4.3 GeneLab Consensus Processing Pipelines

One of GeneLab's most important contributions is the development and publication of standardized consensus bioinformatics pipelines. These pipelines are developed by the GeneLab Analysis Working Groups (AWGs), committees of bioinformatics experts who review and validate best practices for each data type.

For RNA-seq data, the consensus pipeline includes:

  1. Raw data quality check with FastQC and MultiQC
  2. Adapter trimming with Trim Galore
  3. Alignment to reference genome with STAR (using Ensembl reference genomes)
  4. Alignment quality check with RSeQC and samtools
  5. Gene-level count quantification with STARsolo or featureCounts
  6. Normalization and differential expression analysis with DESeq2
  7. GO and pathway enrichment analysis

All pipeline code is publicly available on the GeneLab GitHub repository at github.com/nasa/GeneLab_Data_Processing.

4.6 Data Reuse and Meta-Analysis

One of the most powerful aspects of open-access databases like GeneLab is the ability to perform meta-analysis — combining data from multiple independent studies to identify robust, reproducible findings. Meta-analysis in spaceflight bioinformatics has been particularly impactful because individual studies are necessarily small.

The GeneLab Data Systems platform enables users to: search for all datasets using the same organism, tissue, and study factor combination; download processed count matrices from multiple studies; re-analyze with a consistent pipeline; and use batch correction methods (e.g., ComBat, limma::removeBatchEffect) to harmonize data from different missions.

Session Summary

  • NASA OSDR (incorporating GeneLab) hosts 500+ spaceflight and space-analog studies with standardized metadata, raw data, and processed results
  • Each OSD/GLDS accession contains raw FASTQ files, processed count matrices, DEG results, and ISA metadata tables
  • GeneLab consensus pipelines (GitHub) ensure reproducible, standardized data processing across all datasets
  • GLDS-48, GLDS-242, and GLDS-530 are particularly good starting datasets for learning
  • Meta-analysis across multiple GeneLab datasets increases statistical power for small-sample spaceflight studies
